- **Цель:** Проверка вычислений при a = 0  
- **Входные данные:** a = 0, b = 8, c = 16

- **Описание процесса:** Используется стандартный вызов функции с контролем возвращаемого значения.

- **Ожидаемый результат:** 

  a = 4, b = -12, c = 5: [0.5, 2.5] - список из 2-х элементов;
